== Installation ==

1. Upload the entire `mixforms` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'MixForms' menu in your WordPress admin panel.

For basic usage, you can also have a look at the [plugin homepage](https://mix-forms.com/).

== Shortcodes ==

Use [mixforms key="YOURKEY"] for default usage

Options:
	property		values			default				description
	style 			iframe/embed	embed				loads the form in an iframe or embed it seamlessly in the page
	autoFocus		true/false		true				enable or disable focussing the first element at form-startup (autofocus can cause unwanted pagescroll)
